# caleb-games

A collection of browser-based games for Caleb to play on his tablet. All games are touch-optimized, self-contained HTML files with no external dependencies.

## Games

- **Arcanoid** - Breakout clone with 12 power-ups
- **Tetris** - Classic Tetris with SRS rotation and touch buttons
- **Frogger** - Cross the road and river to reach home
- **Paperboy** - Deliver newspapers while dodging obstacles
- **Tron** - Neon lightcycle battle vs AI
- **Worms** - Turn-based artillery with destructible terrain
- **2048** - Slide and merge tiles to reach 2048
- **Snake** - Eat food, grow longer, don't hit yourself
- **Pac-Man** - Chomp dots and dodge the ghosts
- **Bomberman** - Place bombs and blast your way through
- **Whack-a-Mole** - Tap the moles before they hide
- **Fruit Ninja** - Swipe to slice fruit, avoid the bombs
- **Tower Defense** - Build towers and stop enemy waves
- **Endless Runner** - Jump and duck to survive the run
- **Bubble Shooter** - Aim, shoot, and pop matching bubbles

## Local Development

Requires [serve](https://www.npmjs.com/package/serve) (`npm install` to set up):

```bash
npm install
npm start
```

This runs `serve .` on port 5000 — open http://localhost:5000 in your browser.
